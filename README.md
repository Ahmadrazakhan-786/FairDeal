# 🚀 FairDeal – Local Service Marketplace Platform

FairDeal is a full-stack **MERN** web application where users can find and book trusted local service providers like **Electricians, Tutors, Plumbers, Tailors, Cooks, Painters, Cleaners, and Carpenters**, while providers can register and offer their services.

Built with **React + Node.js + Express + MongoDB** and deployed on **Vercel + Render**.
